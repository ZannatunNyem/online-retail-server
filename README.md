# 🛒 Online Retail Server

## 📖 Project Overview

**Online Retail Server** is the backend API for the Online Retail e-commerce application. It uses **Node.js** and **Express** to provide a structured and secure REST API that allows:

- User authentication
- Cart & Product data delivery
- Request handling for frontend actions

This backend works with the **Online Retail Client** frontend to support:

✔ Login required to add items to cart  
✔ User-specific cart storage 

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **Node.js** | JavaScript runtime for server |
| **Express.js** | Backend routing and middleware |
| **MongoDB** | NoSQL database for storing products, users, and cart data |
| **CORS** | Access control for frontend API calls |
| **dotenv** | Environment variable handling |
| **REST API** | Standard endpoint design |

---

## 📝 Features

✔ User authentication support  
✔ RESTful API routes for products & cart  
✔ Cart management for each user 
✔ Supports secure communication with frontend
